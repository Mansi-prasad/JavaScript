JavaScript HTML DOM EventListener

The addEventListener() method attaches an event handler to the specified element, without overwriting existing event handlers.

We can add many event handlers of the same type to one element,

Don't use the "on" prefix for the event; use "click" instead of "onclick".

Syntax: 

element.addEventListener(event, function, useCapture);

    where,
  • event: can be event type like alick,mousedown etc.
  • function: function we want to call when the event occurs
  • useCapture: a boolean value specifying whether to use event bubbling or event capturing. This parameter is optional.
The default value for useCapture is false, which will use the bubbling propagation, when the value is set to true, the event uses the capturing propagation.
In bubbling the inner most element's event is handled first and then the outer: the p element's click event is handled first, then the div element's click event.
In capturing the outer most element's event is handled first and then the inner: the div element's click event will be handled first, then the p element's click event.

removeEventListener() method

removes event handlers that have been attached with the addEventListener() method
