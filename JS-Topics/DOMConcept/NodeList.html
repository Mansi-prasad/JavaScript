JavaScript HTML DOM Node Lists

A NodeList object is a list (collection) of document nodes (element nodes, attribute nodes, and text nodes).

NodeList items can only be accessed by their index number.

A NodeList is most often a static collection. Example: If you add a li element to a list in the DOM, the list in NodeList will not change.

Browsers return a NodeList object for the method querySelectorAll() and for the property childNodes.

A NodeList may look like an array, but it is not. You can loop through a NodeList and refer to its nodes by index. But, you cannot use Array methods like push(), pop(), or join() on a NodeList.

HTML DOM Node List Length

The length property defines the number of nodes in a node list
