JavaScript Static Methods

Static methods are functions that belong to a class itself,, they do not call by instances of the class. You call them on the class directly, not on objects created from the class.

Advantages of Static Methods

  1. Belong to the class, not individual instances.
  2. Cannot access this referring to instance properties.
  3. Best used for utility or helper functions.
  4. Can be inherited by subclasses.
  5. Called directly with the class name: ClassName.method()
  6. Can only access other static properties or methods of the class.
  7. Cannot override instance methods.
