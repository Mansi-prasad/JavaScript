JavaScript Closures

JavaScript variables can belong to:
The local scope or The global scope

Global variables can be made local (private) with closures. Closures makes it possible for a function to have "private" variables.

Variable Lifetime

Global variables live until the page is discarded, like when you navigate to another page or close the window.

Local variables have short lives. They are created when the function is invoked, and deleted when the function is finished.

Closure makes it possible for a function to have "private" variables.
