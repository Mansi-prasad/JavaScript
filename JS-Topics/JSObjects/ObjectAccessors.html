JavaScript Object Accessors (Getters and Setters)

Getters and setters allow you to define Object Accessors (Computed Properties).

Getter uses the get keyword and Setter uses the set Keyword

differences between Function or Getter?

Function-> access as a function: Object.property().

Getter-> access as a property: Object.property(also provide a simpler syntax).

JavaScript can secure better data quality when using getters and setters.

Why Using Getters and Setters :

  • It gives simpler syntax
  • It allows equal syntax for properties and methods
  • It can secure better data quality
  • It is useful for doing things behind-the-scenes
