Web History API

The Web History API provides easy methods to access the windows.history object(contains the URLs (Web Sites) visited by the user)

  1. The back() method loads the previous URL in the windows.history list.
  2. The go() method loads a specific URL from the history list:
    3. The forward() method Loads the next URL in the history list

History Object Properties

length Returns the number of URLs in the history list
