Web APIs

A Web API is an application programming interface for the Web.(A Browser API can extend the functionality of a web browser, and A Server API can extend the functionality of a web server. )

The position is not available unless the user approves it. and The Geolocation API will only work on secure contexts such as HTTPS.

Browser APIs

All browsers have a set of built-in Web APIs to support complex operations, and to help accessing data. EX: Geolocation API can return the coordinates of where the browser is located.

Third Party APIs

To use these APIs, We will have to download the code from the Web.

Examples:

  • YouTube API - Allows us to display videos on a web site.
  • Twitter API - Allows us to display Tweets on a web site.
  • Facebook API - Allows us to display Facebook info on a web site.
